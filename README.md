# .github

Org-level special repo for [truenas-community-sysexts](https://github.com/truenas-community-sysexts). Holds default community health files (`CONTRIBUTING.md`, the PR template) and the org profile README (`profile/README.md`) that renders on the org's front page.

Files here apply org-wide as defaults — individual repos that ship their own `CONTRIBUTING.md` or `.github/pull_request_template.md` override these.
